# redbox-cli
This is a command line parser


## Installation

Using [composer](https://packagist.org/packages/league/climate):

```bash
$ composer require league/climate
```

The following versions of PHP are supported by this version.

+ PHP 5.4
+ PHP 5.5
+ PHP 5.6
+ HHVM
